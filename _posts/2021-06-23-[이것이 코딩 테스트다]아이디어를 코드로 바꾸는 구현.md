# 아이디어를 코드로 바꾸는 구현

## 피지컬로 승부하기
- 구현 : 머릿속에 있는 알고리즘을 소스코드로 바꾸는 과정
- 완전탐색 , 시뮬레이션 유형
- 완전탐색 : 모든 경우의 수를 주저 업싱 다 계산 하는 해결 방법
- 시뮬레이션 : 문제에서 제시한 알고리즘을 한단계씩 차례대로 직접수행

## 고려해야 할 메모리 제약 사항
- 코딩테스트 메모리 제한 : 128MB ~ 512MB
- 10,000,000 : 약 40MB
- 1초에 2000만 번의 연상 수행 가정 -> 안정적
- 즉 데이터 100만 -> NlogN 이내
